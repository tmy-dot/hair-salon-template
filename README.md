# MASTER TEMPLATE Vol.2 - Hair Salon

美容室・ヘアサロン向けの営業用デモサイトです。

## 変更する場所

主に `index.html` の以下を変更します。

- 店名
- メインコピー
- サブコピー
- コンセプト文章
- メニュー名・価格
- スタッフ名
- 店舗住所
- 営業時間
- 定休日
- 電話番号
- Instagram / LINE / 予約URL
- Google Map

## 画像の差し替え

以下の画像を同じ名前で入れ替えると、別店舗向けに変更できます。

```text
images/hero.jpg
images/concept.jpg
images/gallery/gallery01.jpg
images/gallery/gallery02.jpg
images/gallery/gallery03.jpg
images/gallery/gallery04.jpg
images/staff/staff01.jpg
images/staff/staff02.jpg
```

## 営業での説明

美容室・ヘアサロン向けに制作したデモサイトです。スマートフォン対応、メニュー、ギャラリー、スタッフ紹介、アクセス、予約導線まで実装しています。
